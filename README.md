# CRM Project – Contacts & Meta Chat Integration

This is a **CRM project** designed to manage contacts and enable chat functionality through Meta apps: **Facebook, Instagram, and WhatsApp**.  
It includes features for contact management, filtering, and messaging.

## Getting Started

### 1. Clone the repository

```bash
git clone <YOUR_GIT_URL>
```

### 2. Navigate to the project directory
```bash
cd <YOUR_PROJECT_NAME>
```

### 3. Install dependencies
```bash
npm install
```
### 4. Start the development server
```bash
npm run dev
```

The app will be available at http://localhost:5173
 by default.
